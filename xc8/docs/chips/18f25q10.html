18F25Q10 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// External Oscillator mode Selection bits : EC (external clock) above 8 MHz; PFM set to high power
// Power-up default value for COSC bits : EXTOSC operating per FEXTOSC bits (device manufacturing default)
#pragma config FEXTOSC = ECH, RSTOSC = EXTOSC
#pragma config <setting>=<literal constant>
For example:
// External Oscillator mode Selection bits : EC (external clock) above 8 MHz; PFM set to high power
// Power-up default value for COSC bits : EXTOSC operating per FEXTOSC bits (device manufacturing default)
#pragma config FEXTOSC = 0x7, RSTOSC = 0x7
#pragma config <register>=<literal constant>
For example:
// External Oscillator mode Selection bits : EC (external clock) above 8 MHz; PFM set to high power
// Power-up default value for COSC bits : EXTOSC operating per FEXTOSC bits (device manufacturing default)
#pragma config CONFIG1L = 0xFF
For example:
// IDLOC @ 0x200000
#pragma config IDLOC0 = 0x65535

Configuration Registers & Settings

Register: CONFIG1L @ 0x300000
FEXTOSC External Oscillator mode Selection bits
LP LP (crystal oscillator) optimized for 32.768 kHz; PFM set to low power
XT XT (crystal oscillator) above 100 kHz, below 8 MHz; PFM set to medium power
HS HS (crystal oscillator) above 8 MHz; PFM set to high power
OFF Oscillator not enabled
ECL EC (external clock) below 100 kHz; PFM set to low power
ECM EC (external clock) for 500 kHz to 8 MHz; PFM set to medium power
ECH EC (external clock) above 8 MHz; PFM set to high power
RSTOSC Power-up default value for COSC bits
HFINTOSC_64MHZ HFINTOSC with HFFRQ = 64 MHz and CDIV = 1:1
RESERVED_1 Reserved (DO NOT USE)
EXTOSC_4PLL EXTOSC with 4x PLL, with EXTOSC operating per FEXTOSC bits
SOSC Secondary Oscillator
LFINTOSC Low-Frequency Oscillator
HFINTOSC_1MHZ HFINTOSC with HFFRQ = 4 MHz and CDIV = 4:1
EXTOSC EXTOSC operating per FEXTOSC bits (device manufacturing default)
Register: CONFIG1H @ 0x300001
CLKOUTEN Clock Out Enable bit
OFF CLKOUT function is disabled
ON CLKOUT function is enabled
CSWEN Clock Switch Enable bit
OFF The NOSC and NDIV bits cannot be changed by user software
ON Writing to NOSC and NDIV is allowed
FCMEN Fail-Safe Clock Monitor Enable bit
OFF Fail-Safe Clock Monitor disabled
ON Fail-Safe Clock Monitor enabled

MCLRE Master Clear Enable bit

PWRTE Power-up Timer Enable bit
ON Power up timer enabled
OFF Power up timer disabled
LPBOREN Low-power BOR enable bit
OFF Low power BOR is disabled
ON Low power BOR is enabled
BOREN Brown-out Reset Enable bits
SBORDIS Brown-out Reset enabled , SBOREN bit is ignored
NOSLP Brown-out Reset enabled while running, disabled in Sleep; SBOREN is ignored
ON Brown-out Reset enabled according to SBOREN
OFF Brown-out Reset disabled

ZCD ZCD Disable bit
OFF ZCD disabled. ZCD can be enabled by setting the ZCDSEN bit of ZCDCON
ON ZCD always enabled
PPS1WAY PPSLOCK bit One-Way Set Enable bit
ON PPSLOCK bit can be cleared and set only once; PPS registers remain locked after one clear/set cycle
OFF PPSLOCK bit can be set and cleared repeatedly (subject to the unlock sequence)
STVREN Stack Full/Underflow Reset Enable bit
ON Stack full/underflow will cause Reset
OFF Stack full/underflow will not cause Reset
XINST Extended Instruction Set Enable bit
OFF Extended Instruction Set and Indexed Addressing Mode disabled
ON Extended Instruction Set and Indexed Addressing Mode enabled
