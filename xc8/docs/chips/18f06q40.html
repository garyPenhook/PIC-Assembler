XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// External Oscillator Selection : EC (external clock) above 8 MHz
// Reset Oscillator Selection : EXTOSC operating per FEXTOSC bits (device manufacturing default)
#pragma config FEXTOSC = ECH, RSTOSC = EXTOSC
#pragma config <setting>=<literal constant>
For example:
// External Oscillator Selection : EC (external clock) above 8 MHz
// Reset Oscillator Selection : EXTOSC operating per FEXTOSC bits (device manufacturing default)
#pragma config FEXTOSC = 0x7, RSTOSC = 0x7
#pragma config <register>=<literal constant>
For example:
// External Oscillator Selection : EC (external clock) above 8 MHz
// Reset Oscillator Selection : EXTOSC operating per FEXTOSC bits (device manufacturing default)
#pragma config CONFIG1 = 0xFF
For example:
// IDLOC @ 0x200000
#pragma config IDLOC0 = 0x65535

XT XT (crystal oscillator) above 100 kHz, below 8 MHz
HS HS (crystal oscillator) above 8 MHz
OFF Oscillator not enabled
ECL EC (external clock) below 100 kHz
ECM EC (external clock) for 500 kHz to 8 MHz
ECH EC (external clock) above 8 MHz
RSTOSC Reset Oscillator Selection
HFINTOSC_64MHZ HFINTOSC with HFFRQ = 64 MHz and CDIV = 1:1
EXTOSC_4PLL EXTOSC with 4x PLL, with EXTOSC operating per FEXTOSC bits
SOSC Secondary Oscillator
LFINTOSC Low-Frequency Oscillator
HFINTOSC_1MHZ HFINTOSC with HFFRQ = 4 MHz and CDIV = 4:1
EXTOSC EXTOSC operating per FEXTOSC bits (device manufacturing default)

	void __interrupt(irq(SWINT)) isr0(void) { 
		// do stuff
	}
For example: macro usage. 
	void __interrupt(irq(default)) isr1(unsigned char irq) {

				// do Oscillator Failure Interrupt stuff
				break;
